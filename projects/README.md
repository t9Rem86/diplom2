# Проекты

JSON-файлы параметров проектов складских комплексов.

Каждый файл создаётся командой `/new-project` и содержит:

| Поле | Описание |
|---|---|
| `name` | Название проекта (латиница) |
| `start_date` | Дата начала строительства (ГГГГ-ММ) |
| `area_sqm` | Площадь объекта, кв.м. |
| `construction_cost` | Себестоимость строительства, руб. с НДС |
| `loan_share` | Доля кредита (0–1) |
| `cbr_rate` | Ставка ЦБ |
| `loan_rate` | Кредитная ставка |
| `rental_rate_sqm` | Арендная ставка, руб./кв.м./мес. с НДС |
| `inflation_rate` | Годовой уровень инфляции |
| `construction_months` | Срок строительства, мес. |
| `horizon_months` | Горизонт планирования, мес. |
| `vat_rate` | Ставка НДС (по умолчанию 0.22) |
